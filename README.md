# js-config

Monorepo for all reusable JS config packages

## Setup

Add the following to your `~/.npmrc`

```
@marcdonald:registry=https://npm.pkg.github.com/
//npm.pkg.github.com/:_authToken=YOUR_GITHUB_PERSONAL_ACCESS_TOKEN
```
